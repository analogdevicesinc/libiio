Ptr

template<class obj_T, class ptr_T, void deleter_T>
class Ptr

Special unique pointer for instances that must be destroyed.

Template Parameters:

  • obj_T – Wrapper class

  • ptr_T – Pointer type from the C-API

  • deleter_T – Function that must be used for destroying objects
